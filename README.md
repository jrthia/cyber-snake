# cyber-snake
A cyber themed Snake game with a built-in music player and beat-reactive visuals, inspired by Tetris Effect Connected.

## Features
- Three worlds: Deep Ocean, Outer Space, Neon City
- Built-in music player — drag and drop your own tracks
- Visuals react to music in real time using the Web Audio API
- Beat detection drives border glow, particle bursts and colour shifts

## Notes
- No playlist memory - tracks are loaded one at a time and loop until replaced. Full playlist support will be added in the future.
- Border glow might be too intense and jittery during music. This will be fixed in the future.
